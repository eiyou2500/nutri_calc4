# Visual Thesis

## 1. Experience Thesis
20代女性の利用者が開いた瞬間に、余計な説明を読まされず「すぐ入力できる」「数字が怖く見えない」「でも医療計算として信頼できる」と感じる。

## 2. Visual Thesis
「手帳のように軽く開ける栄養計算シート。淡いコーラルの目印で、入力すべき欄と注意すべき数字だけがやさしく浮く」。

## 3. Motion Thesis
動きは入力欄・候補・警告に触れた瞬間の状態確認だけに使い、計算開始を妨げる演出は置かない。

## 4. Anti-Slop Rules
- 実務前に読ませるランディングヒーローを置かない。
- 紫青の抽象グラデーションやぼかし玉を使わない。
- かわいさのためだけのイラストや意味のない装飾を置かない。
- カードの角丸・影・余白を全部同じにしない。
- 薄いグレー文字だけで情報階層をごまかさない。
