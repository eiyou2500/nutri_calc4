# Visual Thesis

## 1. Experience Thesis
病棟で食事・点滴・経腸・ONSをまたいだ栄養量を急いで確認する人が、5秒後に「入力すべき順番と注意すべき数字が先に見えて、申し送り前の迷いが減りそうだ」と感じる。

## 2. Visual Thesis
「紙の栄養指示簿に、必要な計算欄だけが鉛筆の罫線のように浮かび上がる」画面にする。

## 3. Motion Thesis
動きは装飾ではなく、入力・候補選択・警告・集計という状態変化が起きた場所を静かに示すために使う。

## 4. Anti-Slop Rules
- 紫青の抽象グラデーションやぼかし玉を使わない。
- 意味のないbento gridやダミーロゴを置かない。
- 既存の計算作業を隠す抽象的なヒーローを作らない。
- カードの角丸・影・余白を全部同じにしない。
- 薄いグレー文字だけで情報階層をごまかさない。
